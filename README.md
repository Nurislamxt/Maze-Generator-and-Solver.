# DFS Maze Generator & A* Pathfinding Visualizer

Веб-приложение для генерации лабиринтов и поиска кратчайшего пути. Проект выполнен в рамках экзаменационной работы по дисциплине «Информационные системы и технологии».

---

## 🚀 Описание проекта

Приложение работает полностью в браузере и демонстрирует два классических алгоритма:

- DFS (Depth-First Search) — генерация идеального лабиринта
- A* (A-star) — поиск кратчайшего пути от старта до финиша

Проект наглядно визуализирует работу алгоритмов на Canvas.

---

## ⚙️ Функциональность

- Генерация случайного лабиринта (DFS)
- Поиск кратчайшего пути (A*)
- Визуализация процесса на HTML5 Canvas
- Выбор размера лабиринта (например 10x10, 20x20, 30x30)
- Анимация построения пути
- Кнопки управления: Generate / Solve / Reset
- Темная тема интерфейса

---

## 🛠 Технологии

Проект реализован без фреймворков и сторонних библиотек.

- HTML5 — структура и Canvas
- CSS3 — стилизация и адаптивный интерфейс
- JavaScript (ES6) — логика, алгоритмы, работа с Canvas
